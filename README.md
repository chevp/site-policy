# Site Policy

Central repository for legal and policy pages shared across all chevp sites.

## Pages

- **index.html** - Policy overview and navigation
- **terms.html** - Terms of Service
- **privacy.html** - Privacy Policy (GDPR compliant)
- **impressum.html** - Legal disclosure (Swiss law)
- **contact.html** - Contact information

## Usage

This repository is deployed as GitHub Pages at:

```
https://chevp.github.io/site-policy/
```

Other sites link to these pages for their legal/policy requirements:
- `https://chevp.github.io/site-policy/terms.html`
- `https://chevp.github.io/site-policy/privacy.html`
- `https://chevp.github.io/site-policy/impressum.html`
- `https://chevp.github.io/site-policy/contact.html`

## Deployment

Push to `main` branch triggers automatic GitHub Pages deployment.
